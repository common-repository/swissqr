== Description ==

Mit diesem Plugin kann die QR-Rechnung die ab 30.09.2022 eine Pflichtvorgabe ist mit dem wpShopGermany umgesetzt werden.

- Anzeige des QR Codes und der Zahlungsangaben auf der Bestellabschlußseite
- Optional als PDF Anhang an der Kundenmail nach Bestellung
- Optional als PDF Anhang an der Rechnungsmail (erfordert wpShopGermany Rechnungsmodul)
- Aktivierbar für bestimmte Zahlungsarten Vorkasse/Rechnung etc.

== Installation ==

Zur Installation beachten Sie bitte folgende Schritte:

1. Die Plugindateien nach wp-content/plugins/wpsg_swissqr kopieren, oder über das Wordpress Backend: Plugins > Installieren > Hochladen die ZIP Datei hochladen.
2. Das Plugin im Plugin Menü von Wordpress aktivieren.
3. Die Einstellungen des Plugins erscheinen dann als neues Modul "SwissQR" im wpShopGermany
